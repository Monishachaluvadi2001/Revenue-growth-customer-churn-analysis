# Power BI Dashboards

This folder contains screenshots of the Power BI dashboards created as part of the Revenue Growth and Customer Churn Analysis project.

## Dashboard 1: Revenue Overview
Focuses on overall revenue trends, monthly growth, and key KPIs.

## Dashboard 2: Segment Analysis
Analyzes revenue contribution and trends across customer segments.

## Dashboard 3: Churn Impact Analysis
Highlights churned revenue trends and churn impact over time.
